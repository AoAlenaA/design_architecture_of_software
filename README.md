Компания хочет внедрить единую систему адаптации менеджеров по продажам с помощью курсов сгенерированных ИИ. Для этого нужна разработать аналитический модуль, чтобы прозрачно отслеживать прогресс новичков, быстро находить слабые места и снижать нагрузку на руководителей.

**Пользователи:**  
- Руководителей отделов продаж;  
- HR-специалисты и аналитики;  
- Стажеры- новчики

**Требования:**
- система собирает данные из модулей обучения: результаты тестов, диалоговых тренажёров, итоговых проверок знаний;  
- автоматически выявляет типовые ошибки:
  - пропуск этапов воронки;
  - слабое знание продукта;
  - ошибки в работе с возражениями;
  - проблемы с закрытием сделки;
- формирует понятные отчёты для руководителей:
  - по каждому менеджеру: прогресс адаптации, динамика результатов, проблемные зоны;
  - по команде: общие узкие места, распределение ошибок по этапам воронки;
- поддерживает фильтрацию и визуализацию данных (по периодам, отделам, менеджерам);  
- уведомляет руководителя о рисковых случаях (низкий прогресс, повторяющиеся ошибки).

**Дополнительный контекст:**  
Рынок продаж высоко конкурентен, стоимость найма и обучения менеджеров значительна. Бизнесу важно ускорить окупаемость новичков и сократить объём ручного контроля со стороны руководителей. Система аналитики является частью долгосрочной стратегии по цифровизации HR-процессов и повышению эффективности адаптации персонала.
